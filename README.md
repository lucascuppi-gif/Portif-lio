# Portfólio Premium — Lucas Cuppi

Este projeto é um **portfólio pessoal focado em design editorial e premium**, desenvolvido para destacar competências em engenharia de front-end e estética visual minimalista. Todo o conceito, design e implementação foram realizados inteiramente por mim.

## 🚀 O Projeto

O objetivo central foi criar uma experiência web de alto padrão técnico e visual, demonstrando que é possível entregar interatividade complexa e sofisticação utilizando as tecnologias nativas do navegador.

### 💎 Diferenciais
* **Design & UI:** Identidade visual autoral concebida no Figma, baseada em tipografia forte, uso estratégico de espaço negativo e uma estética inspirada em revistas de design e arquitetura.
* **Desenvolvimento:** Construído exclusivamente com **HTML5, CSS3 e JavaScript puro (Vanilla)**.
* **Interatividade & Motion:** Implementação manual de animações de scroll e manipulação de DOM. O projeto explora o uso avançado de bibliotecas de animação e renderização para elevar a experiência do usuário.
* **Performance:** Código limpo e otimizado para garantir carregamento instantâneo e fluidez total nas transições, sem a dependência de frameworks pesados.

## 🛠️ Tecnologias Utilizadas

* **HTML5 / CSS3:** Estrutura semântica e estilização avançada com Grid e Flexbox.
* **JavaScript (ES6+):** Lógica de interatividade e controle de fluxo.
* **GSAP:** Utilizado para animações complexas de scroll (ScrollTrigger) e manipulação de texto (SplitText).
* **Three.js:** Implementação de elementos 3D (como o modelo do diamante) com renderização via WebGL.
