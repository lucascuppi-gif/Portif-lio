## Portfólio Premium — Lucas Cuppi
Este projeto é um **portfólio pessoal focado em design editorial**, desenvolvido com uma estética minimalista e alto padrão visual.
### 🚀 O Projeto
O objetivo é oferecer uma experiência web sofisticada, utilizando tecnologias nativas para garantir elegância e interatividade fluida.
### 💎 Diferenciais
 * **Design & UI:** Identidade visual autoral baseada em tipografia marcante e uso estratégico de espaços negativos.
 * **Desenvolvimento:** Construído exclusivamente com **HTML5, CSS3 e JavaScript puro (Vanilla)**.
 * **Performance:** Código limpo e otimizado para carregamento instantâneo e transições suaves.
### 🛠️ Tecnologias Utilizadas
 * **HTML5 / CSS3:** Estrutura semântica e layouts avançados com Grid e Flexbox.
 * **JavaScript (ES6+):** Lógica de interatividade e manipulação do DOM.
